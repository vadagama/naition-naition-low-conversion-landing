# Шаблон итогового `hypotheses.md`

Этап 4. Заполни **все** разделы. Пустые секции — ошибка качества.

## Правила текста

- **`label` в таблицах** — продуктовый язык (что увидел пользователь / что помешало). Без `m1`, `m2`, `at_m3_no_interaction` в заголовках и label.
- **`funnelStage`** — технический id (можно в скобках или отдельной колонке).
- **N в кластерах** = сумма N в соответствующих группах воронки. Перед сохранением сверь.
